            LOS BITS
Operaciones con bits:
	Cada digito de ceros y unos tiene como nombre el bit, java tiene una serie de 
    operadores capaces de manipular estos dígitos.

OPERADORES:
	1) A<<B => desplazamiento de A en B posiciones hacia la izquierda

	2) A>>B => desplazamiento de A en B posiciones hacia la derecha, teniendo en cuenta el signo

	3) A>>>B => desplazamiento de A en B hacia la derecha, sin tener en cuenta el signo

	4) A&B => operación AND a nivel de bits

	5) A | B => operación de OR a nivel de bits (puede comparar bits iguales)

	6) A ∧ B => operación de XOR a nivel de bits (no puede comparar bits iguales)

	7) ~A => comportamiento de A a nivel de bits (niega la expresión cambiando el valor de los datos)

    Para operar a nivel de bit es necesario tomar toda la longitud predefinida para el tipo de dato.
            N<<K = n*2∧k
            N>>K = n/2∧k
	
    Los bits se cuentan de derecha a izquierda empezando desde el 0.
